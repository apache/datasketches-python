Density Sketch

class datasketches.density_sketch(k: int, dim: int, kernel: ~_datasketches.KernelFunction = <datasketches.KernelFunction.GaussianKernel object>)

An instance of a Density Sketch for kernel density estimation. Requires a KernelFunction object.

classmethod deserialize(data: bytes, kernel: ~_datasketches.KernelFunction = <datasketches.KernelFunction.GaussianKernel object>)

Reads a bytes object and returns a density sketch, using the provided kerenl or defaulting to a Guassian kerenl
